# OVEA Cosmetics

이 저장소는 정적 HTML/CSS/JS 사이트를 포함합니다.

## GitHub Pages 배포

1. 변경 사항을 커밋하고 `main` 브랜치에 푸시합니다.
2. GitHub 저장소 설정에서 `Pages` 항목으로 이동합니다.
3. 배포 소스를 `gh-pages` 브랜치로 설정합니다.
4. 배포가 완료되면 `https://ohyouha.github.io/OVEACosmetics/` 로 사이트를 확인합니다.

> `index.html`은 VS Code Live Server로만 열면 VS Code가 켜져 있을 때만 동작합니다. GitHub Pages로 배포하면 VS Code를 닫아도 브라우저에서 계속 열립니다.
